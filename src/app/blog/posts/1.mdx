### **프로젝트 목적**

기존의 면접 준비 방식은 정형화된 질문 연습에 한정되어 있어, 실제 면접에서 예상치 못한 질문에 대한 대응력이 부족했습니다.
또한, 면접 경험을 공유하고 피드백을 받을 수 있는 공간이 부족했습니다.
이를 해결하기 위해 **AI 기반 실시간 인터뷰 연습과 커뮤니티 기능을 제공하는 플랫폼**을 구축했습니다.

### **핵심 목표**

- AI 기반 인터뷰 질문 및 피드백 제공
- 실시간 커뮤니티 포럼 기능
- 사용자별 인터뷰 데이터 분석 및 시각화

## **프로젝트 개요**

- **프로젝트명**: HireView
- **설명**: AI를 활용한 인터뷰 연습 및 커뮤니티 기능을 제공하는 웹 애플리케이션
- **배포 링크**: [하이어뷰 바로가기](https://www.hireview.net)
- **깃허브**: [프로젝트 GitHub 링크](https://github.com/rnjstmdals6/hireview-server)
- **작업 인원**: 풀스택 1명(본인) / 디자이너 1명

## **주요 기능**

### **AI 기반 인터뷰 기능**
- 사용자가 입력한 질문을 기반으로 Google Gemini API를 활용하여 AI가 면접 질문을 생성합니다.
- AI가 사용자의 답변을 분석하고 피드백을 제공합니다.
- 스트리밍 응답을 활용하여 자연스러운 인터뷰 흐름을 유지할 수 있도록 구현했습니다.
- AI가 추가적인 꼬리질문을 생성하여 심층적인 면접 연습이 가능하도록 했습니다.

### **커뮤니티 포럼 기능**
- 사용자들이 인터뷰 경험을 공유하고 질문을 주고받을 수 있는 커뮤니티 기능을 제공합니다.
- 실시간 댓글 및 피드백을 받을 수 있도록 구현했습니다.

### **사용자 인터뷰 데이터 분석**
- 면접 연습 기록을 저장하고 사용자의 성과를 분석할 수 있는 기능을 제공합니다.
- 응답 정확도, 키워드 포함 여부, 답변의 일관성 등을 기준으로 분석 데이터를 제공합니다.
- 분석 데이터를 시각화하여 사용자가 본인의 발전 과정을 쉽게 확인할 수 있도록 했습니다.

## **담당 업무**

- **WebClient를 활용한 AI API 연동**: Google Gemini API에 요청을 보내고 응답을 가공하여 클라이언트에 반환하는 기능을 개발했습니다.
- **WebFlux 및 R2DBC 기반 비동기 데이터 처리**: AI 면접 응답을 실시간으로 처리하고, 유저 데이터를 효과적으로 관리하기 위해 WebFlux와 R2DBC를 도입했습니다.
- **커뮤니티 포럼 기능 개발**: 사용자들이 자유롭게 의견을 나눌 수 있도록 게시글 및 댓글 기능을 구현했습니다.
- **인터뷰 데이터 저장 및 분석 기능 개발**: 사용자의 인터뷰 기록을 저장하고, 성과 분석을 위한 데이터 처리 기능을 개발했습니다.

---

## **성과 및 배운 점**

### **WebFlux와 R2DBC를 활용한 비동기 프로그래밍 경험**
- 기존의 동기 방식이 아닌 비동기 데이터 처리를 위해 WebFlux와 R2DBC를 사용했습니다.
- AI 면접 응답을 실시간으로 제공하기 위해 **Flux와 Mono를 반환 타입으로 사용**하며, 함수형 프로그래밍 방식에 익숙해지는 과정이 되었습니다.
- 기존 JPA와 달리 테이블이 자동 생성되지 않기 때문에, **쿼리를 직접 작성하며 데이터 스키마를 관리**해야 했고, 이를 통해 SQL 작성에 익숙해질 수 있는 계기가 되었습니다.

### **WebClient를 활용한 외부 API 연동 경험**
- Google Gemini API를 호출하여 AI 기반 질문을 생성하고 응답을 받아오는 기능을 구현했습니다.
- WebClient를 활용하여 비동기 요청을 처리했으며, **Streaming 응답을 적용하여 인터뷰가 자연스럽게 진행될 수 있도록 개선**했습니다.

## **프로젝트를 통해 성장한 점**

이 프로젝트를 통해 단순히 기능 구현을 넘어서 **비동기 프로그래밍과 트래픽 처리 전략에 대한 실무적인 경험을 쌓을 수 있었습니다.**
- WebFlux와 R2DBC를 적용하면서 기존의 동기 방식과의 차이점을 직접 경험하고, 논블로킹 방식의 데이터 흐름을 이해할 수 있었습니다.
- AI API와의 연동을 통해 **WebClient를 활용한 외부 API 호출 및 응답 처리 방식**을 익혔습니다.

이 프로젝트는 단순한 AI 연동을 넘어서 **비동기 프로그래밍과 실시간 데이터 처리를 경험하며 성장할 수 있었던 중요한 계기**가 되었습니다.
